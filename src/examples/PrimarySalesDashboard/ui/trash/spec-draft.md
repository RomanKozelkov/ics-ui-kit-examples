# Primary Sales Dashboard — Референс

## Предметная область

### Цепочка дистрибуции

```
Производитель → [Primary Sales] → Дистрибьютор → [Secondary Sales] → Аптека/Магазин → [Tertiary] → Покупатель
```

**Производитель** — компания, которая физически создаёт товар (препарат). Владеет рецептурой, производственными мощностями, регистрационными удостоверениями.

**Дистрибьютор** — логистический и финансовый посредник между производителем и аптеками. Берёт на себя:

- хранение товара (склады с температурным режимом)
- доставку в тысячи аптечных точек по всей стране
- товарный кредит для аптек (получили товар сейчас — платят через 30–60 дней)
- таможню, сертификацию, документооборот

Производитель работает с 5–10 дистрибьюторами вместо 60 000+ аптек напрямую — это экономика масштаба.

### Primary Sales

**Primary Sales** — продажи от производителя дистрибьютору. Первое звено цепочки: товар вышел со склада производителя и поступил на склад дистрибьютора.

Именно Primary Sales формирует **выручку производителя**. Дашборд позволяет производителю отслеживать:

- кто из дистрибьюторов растёт / падает
- какие продукты тянут продажи вверх или вниз
- динамику по времени (YoY, помесячно)

### MDLP

**MDLP (Мониторинг Движения Лекарственных Препаратов)** — государственная система маркировки лекарств в России. Один из источников данных на дашборде наряду с собственными данными о продажах.

### Продуктовая иерархия

Типичная иерархия в фарме:

```
Терапевтическая группа
  └── Бренд / МНН (международное непатентованное название)
        └── SKU (например, Амоксициллин 500мг)
```

### Иерархичные слайсеры

**Слайсер** — фильтр, размещённый прямо на дашборде (не в отдельной панели). Нажал — данные срезались.

**Иерархичный слайсер** — слайсер с данными в виде дерева. Пользователь может фильтровать на любом уровне: выбрать всю группу («Антибиотики») или конкретный SKU («Амоксициллин 500мг»).

В текущей реализации слайсеры упрощены до **AS IS dropdown** (плоский список без вложенности).

## Фильтры

### Сценарий

Переключатель сценария анализа:

- **Рост (план Q1)** — текущий активный
- **След. 2025**
- **Неполный 2025**

### Год анализа

Inline-переключатель (или dropdown) выбора года для анализа:

- `2025` / `2024` / `2023`

### Период

Фильтрует диапазон дат внутри выбранного года:

- **YTD** (Year-to-Date) — от начала года по сегодняшнее число
- **QTD** (Quarter-to-Date) — от начала текущего квартала по сегодняшнее число
- **MTD** (Month-to-Date) — от начала текущего месяца по сегодняшнее число

### Атрибуты фактов (иерархичные слайсеры, AS IS — dropdown)

- **Дистрибьюторы** — фильтр по дистрибьюторам
- **Продуктовая иерархия** — фильтр по продуктам / SKU
- **Тип фактов** — MDLP / Sales

### Форматы значений

- **Метрика**: переключатель `Value` / `Units`
- **Валюта** (для Value): переключатель валюты (RUB и др.)

## Элементы с данными

### 1. KPI-карточки (данные за весь выбранный год)

| Карточка               | Влияющие фильтры |
| ---------------------- | ---------------- |
| Primary Sales RUB CY   | Год, Валюта      |
| Primary Sales Units CY | Год              |

Каждая карточка содержит:

- Значение за текущий год (CY)
- Значение за предыдущий год (PY) в виде подписи
- Динамика YoY в % (changesblock)

**Пример (из скриншота):**

- RUB CY: `38.4M ₽` / `+9.0%` / PY (2024): `35.3M ₽`
- Units CY: `2 591 667` / `+4.4%` / PY (2024): `2 483 333`

### 2. Линейный график «Тренд Primary Sales»

Помесячная динамика с YoY%.

**Содержит:**

- Линия CY (текущий год) — синяя
- Линия PY (предыдущий год) — пунктирная серая
- Встроенная столбчатая диаграмма (secondary) — помесячная YoY-динамика в %

**Влияющие фильтры:** все фильтры

### 3. Драйверы роста / падения

Горизонтальная столбчатая диаграмма. Показывает **вклад каждого продукта или дистрибьютора в изменение продаж** (Contribution).

- Зелёный — рост
- Красный — падение
- Переключатель: `Продукты` / `Дистрибьюторы`

**Логика влияния фильтров:**

- Фильтр дистрибьюторов **не влияет** на вид «по Дистрибьюторам»
- Фильтр продуктов **не влияет** на вид «по Продуктам»

### 4. Таблицы «Топ Дистрибьюторы» / «Топ Продукты»

Поля:

| Поле                | Описание                                   |
| ------------------- | ------------------------------------------ |
| `#`                 | Позиция (сортировка по Sales, начиная с 1) |
| Наименование        | Название дистрибьютора или продукта        |
| Units CY / Value CY | Продажи за текущий год                     |
| YOY%                | Динамика vs предыдущий год                 |
| Share               | % от общей суммы                           |
| Rank                | Изменение позиции vs предыдущий год        |

**Логика влияния фильтров:**

- Фильтр дистрибьюторов **не влияет** на таблицу дистрибьюторов
- Фильтр продуктов **не влияет** на таблицу продуктов
- Все остальные фильтры влияют
